# Scripts archivés

Ces fichiers ne sont **pas** nécessaires pour utiliser la base dans Profilarr.

Ils servaient à la migration YAML → PCD, à l’uniformisation des textes FR, ou à des rapports de scores entre profils. Conservés en archive au cas où un mainteneur en aurait besoin.

**Validation officielle du dépôt :** `python3 scripts/validate.py` (3 contrôles essentiels).

**Regénération optionnelle :**

- `scripts/maintain/generate_profile_media_ops.py` → `ops/09`
- `scripts/maintain/generate_cf_tests_sql.py` → `ops/11`
